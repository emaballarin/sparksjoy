# sparksjoy

Utilities, scripts, and tools for Nvidia DGX Spark systems.

## Contents

- **`admin/`** - Administrative scripts for user management and system administration
- **`docker/minilit/`** - Containerized deep learning/reinforcement learning development environment
- **`scripts/`** - System utility scripts for maintenance and optimization
- **`sources/`** - C utilities for system monitoring and resource management

## Quick Start

Each directory contains its own README with specific usage instructions.

## Requirements

- Ubuntu-based DGX system
- Sudo/wheel group membership for administrative tasks
- Docker/Podman for containerized environments
